# OpenClaw VS Code Extension

Adds the branded OpenClaw home experience, configuration editor, and gateway status dashboard inside VSCodium/VS Code.

## Commands
- `OpenClaw: Home` — branded landing panel with install/config shortcuts
- `OpenClaw: Configure` — edit `~/.openclaw/openclaw.json`
- `OpenClaw: Install` — opens a terminal with the correct install script
- `OpenClaw: Status` — live gateway status with start/stop buttons

## Development

```bash
npm install
npm run compile
code . # then F5 to launch Extension Development Host
```
